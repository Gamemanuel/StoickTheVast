## Downloading the Project

* If you are a git user, you can clone the most current version of the repository:

```bash
https://github.com/Gamemanuel/StoickTheVast.git
```

* Or, if you prefer, you can use the "Download Zip" button available through the main repository page.  Downloading the project as a .ZIP file will keep the size of the download manageable.

* You can also download the project folder (as a .zip or .tar.gz archive file) from the Downloads subsection of the [Releases](https://github.com/Gamemanuel/StoickTheVast.git) page for this repository.

Once you have downloaded and uncompressed (if needed) your folder, you can use Android Studio to import the folder

> [!TIP]
> ## Git Functions that are nice
> 
> Remove uncommitted changes in tracked files
> ```bash
> git restore .
> ```
> 
> Remove untracked files and directories
> ```bash
> git clean -fd
> ```
